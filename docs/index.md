# 🎨 PixiePaint

**Ein Malbuch für Kinder ab 3 Jahren.** Komplett offline, ohne Werbung, ohne Datensammlung.

Ausmalen, frei zeichnen, nach Zahlen malen, Buchstaben nachspuren, Sticker kleben, zu zweit auf einem Tablet malen — und alles, was ein Kind malt, bleibt auf dem Gerät.

Für Eltern gibt es bis zu vier Kinder-Profile, eine Elternschranke vor allem, was die App verlässt, eine Sicherung aller Bilder, eine Malzeit-Pause und die Möglichkeit, Töne und Musik abzuschalten. Es gibt keine Werbung, keine In-App-Käufe, keine Konten und keine Datensammlung.

Die App gibt es auf Deutsch, Englisch, Französisch, Italienisch, Niederländisch, Polnisch, Portugiesisch, Spanisch und Türkisch.
